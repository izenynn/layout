# macOS Layout Files

macOS keyboard layout file for NokwtsPlusPlus.

## Files

- `nokwtsplusplus.keylayout` — macOS keyboard layout bundle

## Installation

1. Copy `nokwtsplusplus.keylayout` to `~/Library/Keyboard Layouts/`
2. Log out and back in (or reboot)
3. Add the layout in System Settings > Keyboard > Input Sources
